---------------------------------------------------------------------------------------------------------------------------------

  • JDK or JRE (1.4) must be installed in your windows computer and setup of path and classpath must be completed
  • Access to any of the databases (DB2 UDB, Informix, Ingres, MaxDB, Mimer SQL,Mckoi, Oracle, SQL Server, MySql, PostfreSQL and MS Access) needs to be available
  • Download JDBC driver jar file if not available(Refer :Databases and JDBC Drivers.html)
  • Set the CLASSPATH of JDBC driver in run.bat
  • execute run.bat
  • Enjoy with the sample projects inside Example folder and create new projects
